# Analisis-Efektivitas-Program-BPNT-terhadap-Tingkat-Kemiskinan-dan-Ketahanan-Pangan-di-Indonesia-
Analisis Efektivitas Program BPNT terhadap Tingkat Kemiskinan dan Ketahanan Pangan per Provinsi di Indonesia tahun 2023, dengan menggunakan Proses Data Wrangling. Sumber data yang digunakan yaitu data dari BPS, Open data Jabar, dan Laporan Indeks Ketahanan Pangan tahun 2023
